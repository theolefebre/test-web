<html> = page internet

<head> = info pour le reste de la page

<title> = nom de la page
<meta> = caractère spéciaux (utf)

<body> = contenu visuel principal brut

<div> = sert a faire des trucs avec class
<span> = same shit
<h1-2-3-4-5-6> = titre
<p> = paragraphe
<pre> = variante du <p> a utiliser avec Parci Monie
    id = permet de créer une ancre
<br> = retour a la ligne
<hr> = séparation de contenu
<strong> = gras
<em> = italique
<mark> = surligner
<ul> = liste brut (met des points)
<ol> = liste propre (met des chiffres)
    <li> = element de la liste
<dl> = liste de def
    <dt> = nom de la def
    <dd> = description de la def

<a> = adresse
    href = lien interne/externe - "#" ramène a une ancre - mailto: pour envoyer un mail
    target = permet de choisir où le lien s'ouvre
<img> = png

<!--commentaire--> = commentaire
head et body obligatoires

si t'es pas un connard tu met un seul <h1> par <body>
